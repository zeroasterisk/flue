The Flue ecosystem connects agents and workflows to the systems where work arrives and where applications deploy.

## Connectors

Connectors provide focused integration boundaries for external systems and hosted runtime services. They should remain explicit, composable, and easy to configure per project.

## Deployment targets

Flue supports Node and Cloudflare application targets, with additional deployment guidance and integrations evolving around those deployable artifacts.
